# 🤖 AI Chatbot with Memory

## 📌 Project Overview

AI Chatbot with Memory is a Python-based conversational chatbot powered by Google's Gemini API. The chatbot can remember previous messages during a session by maintaining conversation history, allowing it to provide context-aware responses.

This project was developed as part of the DecodeLabs Generative AI Internship Program.

---

## 🚀 Features

- Interactive command-line chatbot
- Google Gemini API integration
- Conversation memory using Python lists
- Context-aware responses
- Chat history saved to a text file
- Error handling for API issues
- User-friendly terminal interface

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Chatbot-With-Memory.git
```

### Navigate to Project Folder

```bash
cd AI-Chatbot-With-Memory
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 API Key Setup

Open `chatbot.py` and replace:

```python
API_KEY = "YOUR_API_KEY"
```

with your Gemini API key.

---

## ▶️ Run the Project

```bash
python chatbot.py
```

## 💬 Example Conversation

```text
You: What is Python?

Bot: Python is a high-level programming language.

You: Who developed it?

```

The chatbot remembers previous messages during the session and responds based on context.
